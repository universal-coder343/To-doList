# ✅ To-Do List

A clean, minimal task manager built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

🔗 **Live Demo:** [to-do-list-theta-tan.vercel.app](https://to-do-list-theta-tan.vercel.app)

---

## ✨ Features

- **Add Tasks** — Quickly add new tasks via an input field
- **Mark Complete** — Check off tasks as you finish them
- **Live Clock & Date** — Displays the current date and time in the header
- **Task Stats** — Tracks total tasks, completed tasks, and completion rate in real time
- **Active Task Count** — Shows how many tasks are still pending
- **Noise Texture UI** — Stylish background for a modern, minimal aesthetic
- **No Dependencies** — Pure HTML, CSS, and JavaScript

## 🚀 Getting Started

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/universal-coder343/To-doList.git
   cd To-doList
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   open index.html
   ```

No build step or server required — it runs directly in the browser.

## 📊 Stats Panel

The bottom of the app shows three live stats:

| Stat | Description |
|------|-------------|
| **Total** | Total number of tasks added |
| **Completed** | Number of completed tasks |
| **Rate** | Percentage of tasks completed |
